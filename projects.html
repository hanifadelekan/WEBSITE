Crypto Coin Trader

Current Iterations

  • Initial simple moving average based algorithm - This algorithm is a copy of an online available script based on the backtesting library.

    It works by closing any shortselling positions and purchasing equity when the first moving average increases above the second moving average. The first moving average is calculated with fewer days than the second moving average

    Click the link to view the code and run it in your own environment. Please note that you will need to install the various libraries imported at the top of the script. Enter " pip install 'library name' into your terminal to do this. - (Assuming Python is already installed).

  • Standard deviation moving average based algorithm - This is the first alteration of the original algorithm

    It works by closing any shortselling positions and purchasing equity when the distance between moving averages exceed a multiple of the standard deviation of the close prices.

    Click the link to view the code and run it in your own environment. Please note that you will need to install the various libraries imported at the top of the script. Enter " pip install 'library name' into your terminal to do this. - (Assuming Python is already installed).

  • Standard deviation moving average based algorithm V2 - This is the second alteration of the original algorithm

    It works by closing any shortselling positions and purchasing equity when the distance between moving averages exceed a multiple of the standard deviation of the first or second moving averages.

    Click the link to view the code and run it in your own environment. Please note that you will need to install the various libraries imported at the top of the script. Enter " pip install 'library name' into your terminal to do this. - (Assuming Python is already installed).
